二、命令
(1)git init 创建仓库
(2)git status 查看仓库的状态
1. -s表示简略模式
(3)git add 添加到暂存区
1. git add . 将所有文件夹都添加到暂存区
(4)git commit -m “被记录的内容” 提交
(5)git rm --cached 文件名字 将文件从暂存区移除
(6)git log 查看提交记录
1. --oneline  查看简洁的提交记录
(7)git reset 回退提交
1. --soft 回退到某一个版本，保存工作区和暂存区的所有内容
2. --hard回退到某一个版本，丢弃工作区和暂存区的所有内容
3. --mixed回退到某一个版本，保存工作区，丢弃暂存区(默认)
(8)git diff 查看工作区、暂存区、本地仓库之间的差异
1. HEAD表示当前版本，HEAD~n表示上n个版本
(9).gitignore文件
1. 在此文件中加入的内容会被git忽略（可以避免将日志之类的文件添加到仓库里面）
2. 文件夹的格式要用/结尾
3. 生效需要文件不能被添加到版本
(10)git ls-files 展示仓库中的文件
(11)git push -u origin main:main 将本地同步到云端
